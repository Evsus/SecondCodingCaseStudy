getLexeme(consumeopt) → {Lexeme}

Provides sequential access to the sequence of lexemes. Repeatedly calling this method without passing false will return each lexeme in sequence, return null after the last lexeme, then start the sequence over again. Passing false allows the caller to "preview" the next lexeme without advancing past it.
Parameters:
Name Type Attributes Default Description
consume boolean <optional>
 true Indicates whether or not to move the current position in the sequence forward.
Returns:
The lexeme at the current position in the sequence, or null if the sequence has ended.
